## 项目名称
> frp内网穿透服务：暴露内网服务到公网

## 运行条件
> 列出运行该项目所必须的条件和相关依赖
* 条件一：需要内网和公网服务器
* 条件二：写入server、client配置
* 条件三：frps部署在公网服务器，frpc部署在内网服务器

## 运行说明
> 说明如何运行和使用你的项目，建议给出具体的步骤说明
* 操作一：拷贝frpc、frps到/usr/local/bin
* 操作二：配置客户端和服务器请参考frp官网文档
* 操作三：赋予脚本执行权限 chmod +x *.sh后运行，或配置到并使用service服务(service服务配置.md)
